# Project Brief

## Project Name
Yatara Ceylon — Transfer Page Overhaul

## Client
- Company: Yatara Ceylon (Luxury Travel & Transfers, Sri Lanka)
- Contact: Sahan
- Deadline: ASAP
- Budget Range: N/A (owner-developed)

## Problem
The current `/transfers` page looks sterile and lacks premium confidence signals. Airport transfer products are incorrectly mixed into the Journeys/Packages section. The page doesn't compete with elite transfer brands (Blacklane, SIXT ride, Welcome Pickups) in terms of trust, booking confidence, and product depth.

## Goal
Transform the transfer page into an elite, conversion-focused premium transfer booking experience. Clean separation between Transfers and Journeys. Add 22 transfer products across airport, intercity, hourly, and safari categories.

## Primary Users
- International tourists landing at CMB airport
- Luxury travelers moving between Sri Lankan destinations
- Families needing reliable, premium ground transport
- Business travelers needing hourly chauffeur service

## MVP Goal
A visitor can land on `/transfers`, understand the full transfer offering, browse signature routes with pricing, choose a vehicle tier, and request a quote — all within 60 seconds.

---

# Requirements

## Functional Requirements

### 1. Information Architecture Fix
- Transfer products must be separated from Journeys/Packages
- `/packages` page fetches only type: 'journey'
- `/transfers` page uses dedicated transfer data model
- No airport/intercity routes appearing in packages

### 2. Transfer Landing Page Rebuild
New section order:
1. Hero with booking strip
2. What's Included / premium promises (6 icons)
3. Choose Your Transfer Type (5 premium cards)
4. Signature Routes (expanded with price, tier, inclusions)
5. Fleet Tiers (with real image support, use cases)
6. How It Works (4-step process)
7. Transfer FAQ
8. Concierge CTA
9. Footer

### 3. Transfer Data Model Expansion
Add 22 transfer products:
- 10 airport arrivals/departures
- 8 intercity private transfers
- 4 chauffeur reserve / day-use / event products
Each with: title, slug, transferType, pickup, dropoff, duration, distance, startingPriceLkr, vehicleTierRecommended, passengers, luggage, summary, includes, bestFor, heroImage

### 4. Enhanced Signature Route Cards
Each card must show: route name, type badge, time, distance, starting price, best vehicle tier, included services, CTA

### 5. Transfer Detail Pages
Full premium layout per route:
- Hero with route facts
- What's included
- Vehicle options
- Why choose this transfer
- Booking/inquiry block
- Related routes

### 6. Premium Trust Signals
Surface clearly: meet & greet, flight tracking, free waiting time, bottled water, luggage assistance, 24/7 WhatsApp support

## Non-Functional Requirements
- Performance: page load under 3s (SSG)
- Responsive: mobile, tablet, desktop
- SEO: proper metadata per route
- Brand consistency: deep-emerald, antique-gold, off-white palette

## Core Features

### Feature: Booking Strip
- Fields: pickup, dropoff, date, time, passengers, luggage, transfer type (one-way/round-trip/hourly)
- CTA: "Check Fare" → links to /inquire with prefilled params
- Position: directly under hero

### Feature: Transfer Type Cards
- 5 categories: Airport, Intercity, Hourly, Safari, Evening/Event
- Each card: full image, title, one-line pitch, best for, duration, starting price, CTA

### Feature: Expanded Signature Routes
- 22 route products with full data
- Card shows: route, type badge, time, distance, price, tier, inclusions
- Links to detail page

### Feature: Fleet Section
- 3 tiers with real image support
- Use case descriptions per tier
- Passenger/luggage capacity
- Feature lists
